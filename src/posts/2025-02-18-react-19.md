## 前言

React 19 是近年来最重要的版本更新之一。它不仅带来了性能层面的突破，更从根本上改变了我们思考 UI 构建的方式。

## Server Components

Server Components 允许组件在服务端渲染并直接流式传输到客户端，无需 JavaScript bundle。

```jsx
async function BlogPage() {
  const posts = await db.posts.findAll();
  return (
    <ul>
      {posts.map(post => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
}
```

## 总结

React 19 代表着前端框架向「服务端优先」方向的重要转变。
